Discussion 9: July 12th, 2024#

Timing a compuation using NumPy vs base Python#

import numpy as np

rng = np.random.default_rng()

arr = rng.integers(1, 10, size = 10**6)

Look @ first 5 elements of arr, divide 1.5 by each element

arr[:5]

array([2, 6, 8, 2, 8], dtype=int64)

1.5/arr[:5]

array([0.75  , 0.25  , 0.1875, 0.75  , 0.1875])

Can we do the same thing with a list?

mylist = list(arr)

mylist[:5]

[2, 6, 8, 2, 8]

#method 1 -- use a for loop
newlist = []
for x in mylist[:5]:
    newlist.append(1.5/x)
newlist

[0.75, 0.25, 0.1875, 0.75, 0.1875]

#method 2 -- list comprehension
[1.5/x for x in mylist[:5]]

[0.75, 0.25, 0.1875, 0.75, 0.1875]

Timing computations#

%%timeit
1.5/arr[:5]

2.95 μs ± 43.9 ns per loop (mean ± std. dev. of 7 runs, 100,000 loops each)

%%timeit
newlist = []
for x in mylist[:5]:
    newlist.append(1.5/x)

14.4 μs ± 327 ns per loop (mean ± std. dev. of 7 runs, 100,000 loops each)

%%timeit
[1.5/x for x in mylist[:5]]

14 μs ± 305 ns per loop (mean ± std. dev. of 7 runs, 100,000 loops each)

Counting using NumPy#

Goal: Use NumPy to estimate probilities

#importy numpy as np
#rng = np.random.default_rng()

n = 10**7

  • (a) Make a NumPy array of random integers between -50 and 50 inclusive. Make a list of the same numbers.

arr = rng.integers(-50, 51, size=n)

mylist = list(arr)

  • (b) How many of the integers are less than 10?

count = 0
for x in mylist:
    if x < 10:
        count += 1
count

5940615

len([x for x in mylist if x < 10])

5940615

arr < 10

array([False,  True, False, ..., False, False, False])

np.count_nonzero(arr < 10)

5940615

  • (c) At what indices do these integers less than 10 occur?

np.nonzero(arr < 10)

(array([      1,      11,      12, ..., 9999990, 9999994, 9999995],
       dtype=int64),)

  • (d) Make a new array containing only these integers.

newlist = []
for x in mylist:
    if x < 10:
        newlist.append(x)

#[x for x in mylist if x < 10]

arr[arr < 10]

array([-42, -20, -26, ..., -14, -30, -28], dtype=int64)

  • (e) If you pick a random integer from the original array, what is the probability it is less than 10? How long does each method take?

%%timeit
count = 0
for x in mylist:
    if x < 10:
        count += 1
count/n

1 s ± 113 ms per loop (mean ± std. dev. of 7 runs, 1 loop each)

%%timeit
len([x for x in mylist if x < 10])/n

877 ms ± 48.1 ms per loop (mean ± std. dev. of 7 runs, 1 loop each)

%%timeit
np.count_nonzero(arr < 10)/n

11.5 ms ± 316 μs per loop (mean ± std. dev. of 7 runs, 100 loops each)
